position: static;

Um elemento com position: static; não está posicionado de forma especial; é sempre posicionado de acordo com o fluxo normal da página:

Este elemento div tem position: static;

position: relative;

Um elemento com position: relative; está posicionado em relação à sua posição normal:

Este elemento div tem position: relative;

position: fixed;

Um elemento com position: fixed; está posicionado em relação à janela de visualização, o que significa que sempre permanece no mesmo lugar, mesmo que a página seja rolada:

Este elemento div tem position: fixed;

position: absolute;

Um elemento com position: absolute; é posicionado em relação ao ancestral posicionado mais próximo (em vez de posicionado em relação à janela de visualização, como fixo):

Este elemento div tem position: relative;
Este elemento div tem position: absolute;

I am sticky!
